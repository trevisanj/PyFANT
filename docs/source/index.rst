PyFANT - stellar spectral synthesis and tools
=============================================

.. only:: html

Welcome!

PyFANT is a Python interface to the `PFANT <http://trevisanj.github.io/PFANT>`_ stellar
spectral synthesis code, with a varied set of extra tools.
